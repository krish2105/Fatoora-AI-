# AI/OCR Provider Setup (OpenAI)

Fatoora AI uses AI for extracting structured data from uploaded receipts (`src/lib/providers/ai`).

## Local Development
In development, the `MockAIProvider` will return hardcoded extraction data to allow you to build and test the Expense Approval UI without spending API credits.

## Production Setup
To enable real AI extraction:

1. Create an [OpenAI](https://platform.openai.com) account and fund it.
2. Generate an API Key.
3. Add to your `.env` (or Vercel):
```env
AI_PROVIDER=openai
OPENAI_API_KEY=sk-...
```

## Security & Confidence Scores
Fatoora AI **never** auto-approves financial records generated by AI. 
The OpenAI provider returns a `confidenceScore` (0.0 to 1.0). If the score is below `0.75`, the UI forces the user to manually review the extracted data before saving.
